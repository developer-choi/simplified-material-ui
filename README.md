# Material UI
> Material-UI의 핵심 로직을 추출하여 정리하고있는 프로젝트입니다.

## 목적
MUI는 다양한 브라우저 환경(IE11 등)과 엣지 케이스를 지원하기 위해 코드의 복잡도가 높습니다.

그래서, 미리 구현 환경에 맞게 최소화하여 추후 참고 자료로 활용하고자 합니다.

## 배경
단순히 뇌피셜로만 구현할 수도 있겠지만,

**최고점**의 결과물을 만들어내려는 저의 개발 습관에서 비롯되었습니다.

## 기준
원본코드의 이 스펙은 확실하게
1. 없어도 된다 ==> 삭제
2. 잘 모르겠다 (기능, 의도 등이 해석이 안됨) ==> 유지
3. 있어야 한다 ==> 유지 + [forworkchoe](https://github.com/developer-choi/forworkchoe/tree/main/packages/core) 로 이관
